Click2Cash is a modern earning-application built using Firebase and Tailwind CSS.
It includes Login/Signup authentication, Spin to Win, Tasks, Ads earning, Referral system, Realtime balance updates, Withdraw requests, and a full dashboard UI.

This project is created in a single HTML file, combining:

HTML

CSS (via Tailwind + custom styles)

JavaScript (Firebase logic)
